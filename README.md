# 🎮 Guess the Number: Interactive Web Game

A sleek, browser-based logic game where players test their intuition by guessing a hidden number between 1 and 100. Built with a focus on clean design and responsive, real-time feedback.

## 📖 Overview
This project is a modern web adaptation of the classic "Guess the Number" logic game. Originally conceptualized as a console application in C, the core logic has been translated into Vanilla JavaScript and wrapped in a responsive HTML/CSS frontend. The application generates a random target number and guides the user toward the correct answer using dynamic "Higher" or "Lower" visual cues.

## ✨ Features
* **Intuitive Gameplay:** Real-time feedback guides players with helpful text, emojis, and color-coded alerts.
* **Modern UI/UX:** Features a clean, centered card-based design with soft shadows and engaging hover states.
* **Robust Input Validation:** Prevents invalid entries (out-of-bounds numbers or blank submissions).
* **Seamless Replayability:** Automatically handles game-over states and smoothly resets the board.
* **Keyboard Support:** Press `Enter` to quickly submit your guess.

## 🚀 Technologies Used
* **HTML5** - Page structure
* **CSS3** - Styling, Flexbox, CSS Variables
* **Vanilla JavaScript (ES6+)** - Game logic and DOM manipulation

## 🕹️ How to Play
1. The game will randomly pick a number between 1 and 100.
2. Enter your guess in the input box and click **Guess** (or press `Enter`).
3. The game will tell you if the secret number is **Higher** or **Lower** than your guess.
4. Keep guessing until you find the correct number!
5. Check your attempt count to see how well you did. Click **Play Again** to start a new round.

## 🛠️ Setup and Installation
Since this is a front-end only project, no installation or build steps are required!
1. Clone this repository or download the source code.
2. Open the `index.html` file directly in any modern web browser.
3. Enjoy the game!
